# zig-pglite

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run bun-todo.ts
```

Build single executable with Bun

```bash
bun build bunt-do.ts --compile --outfile bun-todo
```

Build single executable with Deno

```bash
deno compile --no-check --allow-read --allow-write todo.ts
```
